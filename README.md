# ShadowPet 🐾

An intelligent pet recommendation platform that matches users with the ideal pet based on their lifestyle, budget, and preferences, while providing detailed care guides, ownership costs, and local availability information.

## Features

* Interactive virtual pet experience
* User-friendly interface
* Responsive web design
* Runs directly in the browser
* No installation required

## Live Demo

Visit the live website:

https://rehan-shadow.github.io/shadowpet-improved/

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Rehan-Shadow/shadowpet-improved.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Usage

Interact with your virtual pet through the website interface and explore its features.
